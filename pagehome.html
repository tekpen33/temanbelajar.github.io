Tentang Website Teman Belajar

1. Tujuan Website

Website Teman Belajar dibuat sebagai media pembelajaran berbasis web yang membantu siswa belajar secara mandiri, terstruktur, dan interaktif.

2. Fungsi dan Manfaat

  • Materi pembelajaran terstruktur
  • Quiz evaluasi pemahaman
  • Presensi siswa
  • Mendorong pembelajaran digital

3. Cara Penggunaan Website Teman Belajar

  • Masuk ke Website (Login)

    Pengguna harus melakukan login (Sign In) terlebih dahulu untuk dapat mengakses seluruh fitur pembelajaran pada website Teman Belajar.

    • Buka halaman login Teman Belajar
    • Masukkan username/email dan password
    • Klik tombol Sign In
    • Jika berhasil, pengguna akan diarahkan ke halaman utama (Home)
  • Mengakses Halaman Home

    Setelah login berhasil, pengguna akan berada di halaman Home yang berisi:

    • Informasi tentang website Teman Belajar
    • Tujuan dan manfaat website
    • Panduan singkat penggunaan website

    Pengguna diminta membaca informasi hingga selesai sebelum memulai kelas.

    Disarankan mengikuti materi secara berurutan agar pemahaman maksimal.

  • Mengisi Presensi Kehadiran

    Presensi dilakukan secara online dengan ketentuan:

    • Nama harus sesuai dengan data pendaftaran
    • NISN wajib sesuai dengan data terdaftar
    • Data yang tidak sesuai tidak akan tercatat sebagai kehadiran

4. Penutup

Website ini dirancang untuk membentuk siswa yang mandiri, kritis, dan bertanggung jawab.
